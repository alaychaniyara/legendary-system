# README #

This repo contains all the assignments and exams that were assigned for ITIS/CS 5180/4180 at UNC Charlotte over the past couple of years.

### What is this repository for? ###

* Provide students with an overview of all the material that is covered in the class
* To set expectation accordingly

### What to note when using this repo? ###

* This is not a complete set of all the assignments, and some of the required APIs might not exist.
* The assignments are created based on the class progress and the latest topics introduced by the framework.
